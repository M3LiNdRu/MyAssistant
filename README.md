# MyAssistant

A personal assistant application for managing expenses, investments, and other personal tasks.

## Tech Stack

- **Backend**: ASP.NET Core 10 Web API with MongoDB
- **Frontend**: Angular with Angular Material UI
- **Deployment**: Azure App Service with GitHub Actions CI/CD

## Features

- 💰 **Expense Management**: Track and categorize expenses with tags
- 📊 **Financial Summaries**: Monthly overviews and spending analysis
- 🎯 **Investment Portfolio**: Track multiple portfolios and market prices (coming soon)

For detailed feature documentation, see [docs/EXPENSES_FUNCTIONALITY.md](docs/EXPENSES_FUNCTIONALITY.md).
